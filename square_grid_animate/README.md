# bpy_grid_array_move

## Simple matrix 3d movie euler rotation

bpy_grid_array_move is a Blender script that allows moving objects in a 3D matrix.
Installation
Blender version 2.9 or 3.0.
Only copy the script to the Blender scripts folder.
